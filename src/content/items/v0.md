## 为什么推荐

v0 专注做一件事：生成高质量的前端 UI 代码。你描述「做一个带搜索框的用户列表页面」，它直接给你可运行的 React + Tailwind 代码，还能预览效果，不满意继续对话调整。

生成的代码质量很高，用的是 shadcn/ui 组件库，生产级别可用。

## 适合谁用

- 前端开发者（快速生成组件原型）
- 设计师（不写代码也能出可用界面）
- 独立开发者（快速搭建产品 UI）

## 有什么坑

- 免费版每月有额度限制
- 主要针对 React 生态，Vue/原生不在最佳状态
- 复杂交互逻辑还是需要手动调整
