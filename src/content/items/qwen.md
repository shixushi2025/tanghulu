## 为什么推荐

通义千问背靠阿里云，模型迭代速度快，Qwen2.5 系列在代码和推理上的表现已经相当不错。对于在阿里云生态里工作的开发者，通过 API 调用通义模型也很方便。

国内直接访问，免费额度较慷慨，适合日常使用。

## 适合谁用

- 阿里云/钉钉用户
- 需要国内可用的 AI 助手
- 开发者（通过 API 使用 Qwen 系列模型）

## 有什么坑

- 对标国际顶级模型还有一定差距
- 产品界面和体验不如豆包精细
