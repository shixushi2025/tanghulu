## 为什么推荐

不像 Copilot 只是补全，Claude Code 是真的在「做事情」。你说「帮我重构这个函数」或者「把这个 bug 修掉」，它会真的去读代码、改代码，然后告诉你改了什么。

在终端里运行，不需要切换到浏览器，对开发者来说这很重要。

## 适合谁用

- 写代码的人（任何语言）
- 不喜欢频繁切换上下文的开发者
- 想用 AI 处理真实任务而不只是问答的人

## 有什么坑

- 需要 Anthropic API key，按量计费
- 复杂任务会消耗比较多 token，注意用量
- 目前是英文界面，但支持中文交流
